# Bilibili Study Helper Script

A dedicated auxiliary script designed for online course learning on Bilibili's web version, aiming to optimize the learning experience and boost efficiency.


## âœ¨ Key Features

- **Flexible Playback Speed**  
  Supports custom speed adjustment (0.5x-2.0x) to match different course rhythms and learning habits.

- **One-Click Recommendation Cleanup**  
  Automatically hides irrelevant recommendations on the homepage and playback page, reducing distractions.

- **Full-Screen Chapter Tracking**  
  Displays current chapter title in full-screen mode, keeping you updated on progress without exiting full screen.
